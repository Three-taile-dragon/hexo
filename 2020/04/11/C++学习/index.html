数据类型

整形

作用: 整形变量表示的是整数类型的数据

C++中能表示整型的类型有以下几种方式，区别在于所占用的内存空间不同

数据类型 占用空间 取值范围
short (短整型) 2字节 (-2^15~2^15-1)
int (整型) 4字节 (-2^31~2^31-1)
long (长整型) Windows:4字节，Linux为4字节(32位)，8字节(64位) (-2^31~2^31-1)
long long (长长整型) 8字节 (-2^63~2^63-1)

sizeof关键字：可以求整数所占用的内存空间大小
